# NEAR-MARKETPLACE-CONTRACT
This is a practice for NEAR101 Development course in [decade.org](https://dacade.org/communities/near/courses/near-101)


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Tech Stack


**Blockchain:** NEAR, AssemblyScript
